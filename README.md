
# Adsorption Capacity Prediction using ANN

This project predicts adsorption capacity (%) using an Artificial Neural Network (ANN).

## Files
- train_model.py → Trains model and saves adsorption_ann.h5
- app.py → Runs Gradio web app for predictions
- all_datacsv.csv → Dataset
- adsorption_ann.h5 → Trained model
- requirements.txt → Required Python libraries
